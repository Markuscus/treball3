Sobre Samba

Samba és la suite estàndard d'interoperabilitat de Windows per a Linux i Unix.

Samba és programari lliure amb llicència de GNU General Public License, el projecte Samba és membre de Software Freedom Conservancy.

Des de 1992, Samba ofereix serveis d'impressió i fitxers segurs, estables i ràpids per a tots els clients que utilitzen el protocol SMB/CIFS, com totes les versions de DOS i Windows, OS/2, Linux i molts altres.

Samba és un component important per integrar perfectament servidors i escriptoris Linux/Unix als entorns Active Directory. Pot funcionar tant com a controlador de domini com com a membre normal del domini.
